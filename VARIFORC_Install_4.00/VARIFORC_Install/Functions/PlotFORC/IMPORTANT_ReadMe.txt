IMPORTANT INFORMATION!

The parameter file VARIFORC_PlotFORC_parameters.txt contains default options for plotting whole FORC matrices with a default
color scale (Iridescent), 10 automatically spaced contours, red contour delimiting significant regions of the FORC diagram,
and PDF export format.

Options should be adapted to specific requirements for best results: see the VARIFORC User Manual and the VARIFORC Quick Guide.
The style of almost every element of the output FORC diagram can be controlled by options in the parameter file.
